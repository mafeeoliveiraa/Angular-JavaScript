Introdução sobre os frameworks

Os frameworks são de suma importância para o mundo do trabalho, sendo assim, eles permitem os usuários utilizar soluções existentes sem ter que começar seu código do zero. Dessa maneira, o trabalho do usuário dessa ferramenta é melhor aproveitado, sendo uma ferramenta facilitadora. Dentre os benefícios da utilização de frameworks, estão presentes: mantém um estilo de codificação coerente, utilização de base de códigos, auxiliam na implantação de um gerenciamento estruturado, estabelecem padrões de desenvolvimento, entre outros.

Instalação

Para instalar o Angular framework JavaScript, você precisa instalar o Node.js, o npm e a CLI do Angular. Instalação do Node.js e npm Você pode instalar o Node.js e o npm usando o NVM (Node Version Manager), o gerenciador de pacotes oficial do seu sistema operacional Ou, você pode instalar a partir do site oficial da web Instalação da CLI do Angular Abra o Prompt de Comando Execute o comando npm install -g @angular/cli A saída do comando deve ser a versão do angular-cli instalada Verificação da instalação Para verificar a instalação e a versão atual, execute o comando ng v Para verificar a versão do Angular instalada globalmente, execute o comando npm list -g @angular/cli no terminal.
